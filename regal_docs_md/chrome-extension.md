# Google Chrome Extension

This guide covers how to set up the Regal Chrome Extension, which brings key communication and task management features—like calling, texting, emailing, and status updates—into a compact version of the Regal platform that runs alongside your existing tools. Designed for agents working across multiple tabs in a homegrown or third-party system, the extension helps streamline workflows without requiring a full Regal embed.

> ## 📘
>
> Who is this option best for?
>
> The Chrome Extension is ideal for agents working across multiple tabs within a third-party or homegrown CRM or ticketing system, where embedding Regal directly isn’t practical. The extension offers a streamlined version of Regal’s platform—enabling agents to call, text, email, and manage tasks and contacts—without disrupting their workflow (i.e. switching or refreshing tabs during a call).

# Setup

### Step 1

Enable the Regal Chrome Extension [here](https://chromewebstore.google.com/detail/regalai/odagobonafifelecdclamaickgmnjehi) by clicking 'Add to Chrome'.

![](https://files.readme.io/ec9712f391168377b68475ea6cf9ba3431b22f4f37f831254ed6429bdab83c57-Screenshot_2025-06-20_at_1.31.21_PM.png)

### Step 2

For quick access, you can optionally pin the extension to your browser toolbar. After selecting the puzzle piece, find the Regal.ai extension and click the pin icon.

![](https://files.readme.io/0ffa683f4d6621e676f3bb0b55d427983f4141589a1c4d6e03f669660ec118f7-Screenshot_2025-06-16_at_4.40.41_PM.png)

### Step 3

Lastly, enable microphone permissions by right clicking on the Regal icon in your toolbar and selecting 'View Web Permissions'. You can also get here via the puzzle piece dropdown by selecting 'More Options'.

![](https://files.readme.io/ce45c960edc381fb8dfda0d3ca2965bb6901b90805cab727eecaf8f457b998b8-Screenshot_2025-06-16_at_4.48.32_PM.png)
![](https://files.readme.io/1ff522ec1d1e1de78cfba306a15281b7cf47faeb8c3f607491b007a48f2377d5-Screenshot_2025-06-16_at_4.45.50_PM.png)
  
  
  
  
  
  
  

In the Microphone section, update the permission to 'Allow'.

![](https://files.readme.io/9410b4f5a9ef54a9f0ba371309fe486876c0a4d68cba24610bf7ea1f174d861b-Screenshot_2025-06-16_at_3.40.28_PM.png)

Now you're all set to start using Regal in a Chrome Extension 🎉

![](https://files.readme.io/7727debc84e207cac89f93e492aeb33b3a6281b17455d220ebf00c713a25aa6f-Screenshot_2025-06-16_at_3.43.58_PM.png)

If you usually log into Regal via username and password or Google Single Sign-On, you can continue to do so in the extension. If you usually log in from a third provider, note that you may need to manually open the extension after following that authentication flow (it won't auto-open like the Regal app).

> ## 🚧
>
> Note to avoid calls dropping
>
> Do not close the extension window while on an active call as this will lead the call to drop.
>
> It is not encouraged to use the Regal extension alongside the app in another tab as this could lead calls to drop.
