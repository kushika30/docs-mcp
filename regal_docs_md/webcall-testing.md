# Test Audio (Web calls)

Test the voice quality and conversational handlings of your Voice AI Agents by clicking "Test Audio" to place a web call through the Regal builder.

Web calls are useful for testing your agent while building, because they let you test your agent before hooking up a phone number and routing. However, they do not support testing all functionality, and final E2E testing should be done through actual phone calls.

## ✅ What you **can** test with web calls

![](https://files.readme.io/f94ca365e8cd48d0f8c9dd7455f47b35106e67531bd18d934468b687f7015997-image.png)

- Sound, speed, volume of the voice
- Responsiveness and interruption handling of your agent
- Conversational logic e.g., how it follows your task under different scenarios or objection/question handling or RAG responses
- Custom actions
- Personalization using test values for dynamic variables

## ❌ What you **cannot** test with web calls

- Call transfers
- E2E latency over telephony
- Personalization from actual profile data
- Routing
- Dispositions
