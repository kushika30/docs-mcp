# Search Across Transcripts

To support easier QA of what customers are saying on calls, the Transcription Search dashboard is available search through *contact* transcript moments to quickly find the moments with the best ***semantic* match**. This means the dashboard allows you to search by meaning, beyond traditional exact match and even fuzzy match capabilities.

E.g. "that price is too high" will likely show up as a result for search query "too expensive", even though they have very little word match.

**Where can you find this dashboard?** Under Conversation Intelligence > Insights, you can see Regal Improve - Transcription Search as the 5th dropdown option.
