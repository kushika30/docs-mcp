# How to Add an Agent to a Specific Team

### Teams, like Skills, can help determine which tasks an Agent is eligible to be assigned.

You're able to see which Teams an agent is a member of on the Agents Page.

#### Teams Column

![Teams Column](https://files.readme.io/82bfac6-team.png "team.png")

To change the Teams that an Agent belongs to, click on their name on the Agents Page to open the Agent Slideout. Click on the **Team** dropdown to view the available Teams and select the Team(s) you'd like the Agent to be a part of. Click **Save** to confirm your changes once finished.

#### Teams Dropdown

                                                        ![Screen_Shot_2022-05-24_at_9.33.41_PM.png](https://support.regal.ai/hc/article_attachments/6168029112091/Screen_Shot_2022-05-24_at_9.33.41_PM.png)  

To add or update the list of teams available in your account, email support@regalvoice.com.
