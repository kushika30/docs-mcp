# How to Use Journey Tags

#### **Tags in Journeys**

Organize your Journeys by adding tags to them. To see all relevant Journeys with particular tags, you can filter for the tags in Journeys table.

![](https://support.regal.ai/hc/article_attachments/18050639344795)

![](https://support.regal.ai/hc/article_attachments/18082184758939)

#### **Adding Tags to Existing Journeys**

A Journey can have multiple tags. Type in drop-down the tags you want to add then click to select. To create a new tag, simply type in the tag name and click the "Create Tag" button. Press **Submit** to finalize the changes. There is no need to separately Save Journey as Draft/Live to save the Journey Details information.

![ezgif.com-video-to-gif (3).gif](https://support.regal.ai/hc/article_attachments/18082630186395)

#### **Adding Tags to New Journeys**

Tags can only be added to existing Journeys. Save the new Journey first as Draft or Live before proceeding to add tags.
