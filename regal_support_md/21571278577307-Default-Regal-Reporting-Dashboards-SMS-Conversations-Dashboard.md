# Default Regal Reporting Dashboards: SMS Conversations Dashboard

This guide provides an in-depth exploration of the SMS Conversations Dashboard, one of Regal's default reporting dashboards.

This dashboard only shows data on agent SMS tasks, not automated SMS messages that are sent to contacts. For automated messages, please view the [Campaigns Dashboard](https://support.regal.io/hc/en-us/articles/21564512928027-Default-Regal-Reporting-Dashboards-Campaigns-Dashboard).

While potential use cases for these metrics will be shared, it's essential to emphasize that they should be considered alongside other data sources and tailored to specific organizational goals.

Reporting dashboards are updated: Monday-Friday: Hourly from 9am ET to 10pm ET and Saturday-Sunday at 8am, 3pm, 8pm, and 10pm ET.

Click on the tile or filter name below to learn more about that tile.

# Filters

All default Regal dashboards have filters available for you to refine the data you're viewing. The SMS Conversations Dashboard has the following filters:

![Screenshot 2023-12-27 at 3.44.12 PM.png](https://support.regal.ai/hc/article_attachments/21571278451867)

**Select Timeframe (ET)**

- Used to filter data within a specified timeframe.
- Defaults to "is in the last 7 days".

**Select Period**

- Used to show different periods of time (hourly, daily, weekly, etc.) in the various graphs.
- Defaults to "Daily".

**Day of Week**

- This filter enables you to view data based on specific days of the week.
- Defaults to "is any value".

**Hour of Day (EST)**

- This filter enables you to view data based on specific hours of the day.
- Defaults to "is any value".

**Target Agent**

- Used to filter by the [target agent](https://support.regal.io/hc/en-us/articles/21602595155867-Glossary-of-Regal-Reporting-Terms) assigned to the tasks being viewed.
- Defaults to "is any value".

**Target Team**

- Used to filter by the team of the [target agent](https://support.regal.io/hc/en-us/articles/21602595155867-Glossary-of-Regal-Reporting-Terms) assigned to the tasks being viewed.
- Defaults to "is any value".

**Handling Agent**

- Used to filter by the [handling agent](https://support.regal.io/hc/en-us/articles/21602595155867-Glossary-of-Regal-Reporting-Terms) that worked the tasks being viewed.
- Defaults to "is any value".

**Handling Team**

- Used to filter by the team of the [handling agent](https://support.regal.io/hc/en-us/articles/21602595155867-Glossary-of-Regal-Reporting-Terms) that worked the tasks being viewed.
- Defaults to "is any value".

**Sub Brand**

- Used to filter by phone line used for tasks. You can assign a sub brand to each phone number in Settings > Phone Numbers.
- Defaults to "is any value".

**Brand phone line name**

- Used to filter by phone line used for tasks. You can assign a name to each phone number in Settings > Phone Numbers.
- Defaults to "is any value".

**Task Created Queue Name**

- This filter allows you to filter data by the queue the task was created in.
- Defaults to "any value".

**Task Completed Queue Name**

- This filter allows you to filter data by the queue the task was [completed](https://support.regal.io/hc/en-us/articles/21602595155867-Glossary-of-Regal-Reporting-Terms) in.
- Defaults to "any value".

# Tiles

**Created SMS Tasks**

![Screenshot 2023-12-28 at 3.05.08 PM.png](https://support.regal.ai/hc/article_attachments/21601953775771)

- **Type:** Numeric Display
- **Function:** This tile tracks the total number of agent SMS tasks, regardless of [direction](https://support.regal.io/hc/en-us/articles/21602595155867-Glossary-of-Regal-Reporting-Terms), created during the selected timeframe. When used alongside other tiles, this can be useful for evaluating the reach of your services as well as the efficiency of your agents.

**Completed SMS Tasks**

![Screenshot 2023-12-28 at 3.06.01 PM.png](https://support.regal.ai/hc/article_attachments/21601994302235)

- **Type:** Numeric Display
- **Function:** This tile tracks the total number of agent SMS tasks [completed](https://support.regal.io/hc/en-us/articles/21602595155867-Glossary-of-Regal-Reporting-Terms) by agents during the selected timeframe. When used alongside other tiles, this can be useful for evaluating the reach of your services as well as the efficiency of your agents.

**SMS Task Completion Rate**

![Screenshot 2023-12-28 at 3.08.07 PM.png](https://support.regal.ai/hc/article_attachments/21601994310555)

- **Type:** Numeric Display
- **Function:** This tile displays the percentage of agent SMS tasks [completed](https://support.regal.io/hc/en-us/articles/21602595155867-Glossary-of-Regal-Reporting-Terms) by agents during the selected timeframe. When used alongside other tiles, this can be useful for evaluating the reach of your services as well as the efficiency of your agents.

**SMS Conversations**

![Screenshot 2023-12-28 at 3.11.44 PM.png](https://support.regal.ai/hc/article_attachments/21601953820187)

- **Type:** Numeric Display
- **Function:** This tile tracks the total number of agent SMS tasks [completed](https://support.regal.io/hc/en-us/articles/21602595155867-Glossary-of-Regal-Reporting-Terms) by agents, where a conversation happened with the contact, during the selected timeframe. When used alongside other tiles, this can be useful for evaluating customer engagement levels as well as campaign effectiveness.

**Unique Contacts Had SMS Conversation**

![Screenshot 2023-12-28 at 3.44.05 PM.png](https://support.regal.ai/hc/article_attachments/21602127305883)

- **Type:** Numeric Display
- **Function:** This tile displays the total number of individual contacts that your agents have spoken to via SMS during the selected timeframe. This number will be less than the total number of [conversations](https://support.regal.io/hc/en-us/articles/21602595155867-Glossary-of-Regal-Reporting-Terms) because one contact may have had multiple conversations. When used alongside other tiles, this can be useful for evaluating customer engagement levels as well as campaign effectiveness.

**% SMS Tasks Accepted within 5 Minutes**

![Screenshot 2023-12-28 at 3.52.36 PM.png](https://support.regal.ai/hc/article_attachments/21602419449371)

- **Type:** Numeric Display
- **Function:** This tile displays the percentage of agent SMS tasks (regardless of direction) [accepted](https://support.regal.io/hc/en-us/articles/21602595155867-Glossary-of-Regal-Reporting-Terms) by agents within 5 minutes of the task reservation, during the selected timeframe. When used alongside other tiles, this can be useful for evaluating the efficiency of your agents as well as managing customer expectations regarding wait times and overall service speed.

**Avg. Minutes to Accept SMS Tasks**

![Screenshot 2023-12-28 at 4.26.37 PM.png](https://support.regal.ai/hc/article_attachments/21603470508955)

- **Type:** Numeric Display
- **Function:** This tile shows the average time in minutes it takes for agents to [accept a reservation](https://support.regal.io/hc/en-us/articles/21602595155867-Glossary-of-Regal-Reporting-Terms) for an SMS. Monitoring this average time helps in identifying potential delays in agent processes and can guide agent training. This metric is also important for managing customer expectations regarding wait times and overall service speed.

**Median Minutes to Accept SMS Tasks**

![Screenshot 2023-12-28 at 4.29.07 PM.png](https://support.regal.ai/hc/article_attachments/21603464121115)

- **Type:** Numeric Display
- **Function:** This tile shows the median time in minutes it takes for agents to [accept a reservation](https://support.regal.io/hc/en-us/articles/21602595155867-Glossary-of-Regal-Reporting-Terms) for an SMS. Unlike the average, the median provides a measure that is less influenced by extreme values, offering a clearer view of typical call handling times. Monitoring this metric helps in identifying potential delays in agent processes and can guide agent training. This metric is also important for managing customer expectations regarding wait times and overall service speed.
