# How to Use Saved Views

# Introduction

**Saved Views** allow you to save filtered views of the **Agents**, **Queues**, and **Tasks**pages in the Regal app. You're able to save multiple views on each page and even set one as your default, saving you time and letting you find the information you're looking for quicker on these pages.

# Creating Saved Views

To create a Saved View, apply filters and search criteria to a table like you normally would. Click "Save As" after applying your filters to give the Saved View a name and save it.

![create_saved_view.gif](https://support.regal.ai/hc/article_attachments/18166259691931)

# Editing, Deleting, and Favoriting Views

You're able to edit, delete, or favorite a Saved View after creation using the icons in the dropdown menu. Favoriting a Saved View will make it the default; whenever you visit that page in the Regal app, the default Saved View will be the first thing shown to you.
