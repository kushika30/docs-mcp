# How to Send Automated SMS Messages

### The Send SMS block sends an automated sms campaign.

You must first [create the SMS campaign](https://support.regal.ai/hc/en-us/articles/5721570540699) from the campaign builder. Then, once you've selected the Send SMS block, you'll see a dropdown of SMS campaigns to choose from.

![Send SMS](https://files.readme.io/2f70b8c-sms.png "sms.png")

> ### 📘 SMS Campaigns Must be in Ready State to Send
>
> An SMS campaign must be Saved as Ready in order to trigger it through a journey. If it is in Draft state, the SMS message will not send and the next block in the journey will be processed.
