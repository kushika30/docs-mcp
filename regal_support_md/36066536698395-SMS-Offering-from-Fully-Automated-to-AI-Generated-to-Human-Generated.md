# SMS Offering: from Fully Automated to AI Generated to Human Generated

Texts can be sent as part of automated campaigns, inbound/outbound SMS tasks, or sent manually.

**Automated SMS Campaigns (no AI or Human involvement):**

- Event-driven SMS campaigns let you craft and send personalized, automated text messages to your customers programmatically. Use [this guide](https://support.regal.ai/hc/en-us/articles/5721570540699) to set up an automated campaign and then use a [Send SMS node](https://support.regal.ai/hc/en-us/articles/5724727170715-How-to-Send-Automated-SMS-Messages) in a journey to send the automated text campaign to qualified contacts.
  - For an idea on how to use SMS in your journeys, check out this [Use Case](https://support.regal.ai/hc/en-us/articles/18747177205147-How-to-Create-a-Re-engagement-Journey-SMS) on using journeys and campaigns in driving Re-Engagement.
- [SMS Blasts](https://support.regal.ai/hc/en-us/articles/15178608209435-How-to-Set-Up-an-SMS-Blast) can be set up by setting up a scheduled journey targeting a specific segment.

**SMS That Create Tasks on the Agent Desktop for Human Agents:**

- [Inbound SMS Tasks](https://support.regal.ai/hc/en-us/articles/5052285733531-How-to-Work-with-SMS-Tasks#:~:text=Respond%20to%20an%20Inbound%20SMS) are generated when a contact sends a text to one of your numbers. These tasks can then be routed to the appropriate queues and agents using your [Routing Rules](https://support.regal.ai/hc/en-us/articles/12181758397723-How-to-Configure-Task-Routing-Rules).
- [Outbound Agent SMS Tasks](https://support.regal.ai/hc/en-us/articles/19856872590363-How-to-Create-Agent-SMS-Task-through-Journeys-and-Campaigns) can be generated through Journeys and Campaigns and routed to agents using [Routing Rules](https://support.regal.ai/hc/en-us/articles/12181758397723-How-to-Configure-Task-Routing-Rules). These tasks are triggered by journeys at the right moment and suggest copy to Humana agents to send, but can be modified by agents before sending.
- [Text conversations can be initiated manually](https://support.regal.ai/hc/en-us/articles/5052285733531-How-to-Work-with-SMS-Tasks#:~:text=Literally%2C%20at%20anytime.-,Start%20an%20SMS%20Conversation,-From%20Contact%20Search) at any time by agents from the Agent Desktop.
  - [SMS Templates](https://support.regal.ai/hc/en-us/articles/21244458574491-How-to-Create-Message-Templates) can be created to facilitate agent efficiency and quality. Agents can then [select from created templates](https://support.regal.ai/hc/en-us/articles/21246218602907-How-to-Compose-a-Message-with-Templates) when working any text tasks.
- Text conversation reporting is available through the [default SMS Conversations dashboard](https://support.regal.ai/hc/en-us/articles/21571278577307-Default-Regal-Reporting-Dashboards-SMS-Conversations-Dashboard).

**AI SMS Agent:**

- For the perfect balance between pure automation and manual human effort, let [AI SMS Agents](https://support.regal.ai/hc/en-us/articles/36646480285339-Setting-Up-An-SMS-Agent) craft and send personalized, relevant text messages.

Note: Most customers combine automated, AI-generated, and human-generated SMS strategies depending on use case and quality vs. cost tradeoffs.

For more information on SMS error codes, consult [this page](https://support.regal.ai/hc/en-us/articles/29901230349339-Why-are-my-SMS-not-getting-delivered).

For information about registering for A2P to be able to send text messages consult [this page](https://support.regal.ai/hc/en-us/articles/47587669665691-A2P-Registration-Required-Registration-for-SMS-Messaging-through-10DLC-Numbers).
