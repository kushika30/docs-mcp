# Custom Activity Statuses

By default, Regal only provides two statuses: **Available** and **Unavailable.**However, more options can be added according to your needs. Admins can request to add custom statuses such as "Break" or "Training", for example, by submitting a [Support ticket](https://support.regal.ai/hc/en-us/requests/new).
