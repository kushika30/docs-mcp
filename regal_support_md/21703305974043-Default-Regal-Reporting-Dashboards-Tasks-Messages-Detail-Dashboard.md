# Default Regal Reporting Dashboards: Tasks & Messages Detail Dashboard

This guide provides an in-depth exploration of the Tasks & Messages Detail Dashboard, one of Regal's default reporting dashboards.

While potential use cases for these metrics will be shared, it's essential to emphasize that they should be considered alongside other data sources and tailored to specific organizational goals.

Reporting dashboards are updated: Monday-Friday: Hourly from 9am ET to 10pm ET and Saturday-Sunday at 8am, 3pm, 8pm, and 10pm ET.

Click on the tile or filter name below to learn more about that tile.

# Filters

All default Regal dashboards have filters available for you to refine the data you're viewing. The Tasks & Messages Detail Dashboard has the following filters:

![Screenshot 2024-01-02 at 4.11.10 PM.png](https://support.regal.ai/hc/article_attachments/21704282572699)

**Created At Date**

- Used to filter data by the timeframe in which tasks were created.
- Defaults to "is in the last 7 days".

**Completed At (ET) Date**

- Used to filter data by the timeframe in which tasks were [completed](https://support.regal.io/hc/en-us/articles/21602595155867).
- Defaults to "is null or not null".

**Task Status**

- Used to filter by the current [status](https://support.regal.io/hc/en-us/articles/21219335877147) of the task.
  - Note: Reporting dashboards are not live and if you're looking for a specific task the status may not be the same as it is on the tasks page.
- Defaults to "is any value".

**Task Type**

- Used to filter by the [type of task](https://support.regal.io/hc/en-us/articles/4845317122587).
- Defaults to "is any value".

**Direction**

- Used to filter by the [direction](https://support.regal.io/hc/en-us/articles/21602595155867) of the task.
- Defaults to "is any value".

**Disposition**

- Used to filter by the disposition the task was given.
  - Note: By definition, tasks with a disposition are completed so this will also filter by Task Status = Task Completed.
- Defaults to "is any value".

**Call Campaign ID**

- Used to filter by the campaign ID. Restricted to call tasks.
- Defaults to "is any value".

**Channel**

- Used to filter by the [channel](https://support.regal.io/hc/en-us/articles/21602595155867) of the task.
- Defaults to "is any value".

**Target Agent**

- Used to filter by the [target agent](https://support.regal.io/hc/en-us/articles/21602595155867) assigned to the tasks.
- Defaults to "is any value".

**Agent Email**

- Used to filter by the [handling agent](https://support.regal.io/hc/en-us/articles/21602595155867) of the tasks.
- Defaults to "is any value".

**Target Team**

- Used to filter by the team of the [target agent](https://support.regal.io/hc/en-us/articles/21602595155867) assigned to the tasks being viewed.
- Defaults to "is any value".

**Contact Phone**

- Used to filter by the contact phone. Useful if you want to see all tasks for a specific contact.
- Defaults to "is any value".

**SMS Campaign ID**

- Used to filter by the campaign ID. Restricted to SMS tasks.
- Defaults to "is any value".

**Task Created Queue Name**

- This filter allows you to filter data by the queue the task was created in.
- Defaults to "any value".

**Task Completed Queue Name**

- This filter allows you to filter data by the queue the task was [completed](https://support.regal.io/hc/en-us/articles/21602595155867) in.
- Defaults to "any value".

# Tiles

**Created Call Tasks**

- **Type:** Table
- **Function:** This tile serves as a detailed record of all call interactions, which is useful for tracking, analysis, and historical reference.

**All Created Tasks**

- **Type:** Table
- **Function:** This tile serves as a detailed record of all interactions, regardless of [channel](https://support.regal.io/hc/en-us/articles/21602595155867), which is useful for tracking, analysis, and historical reference..

**All SMS Messages**

- **Type:** Table
- **Function:** This tile serves as a detailed record of all SMS interactions, which is useful for tracking, analysis, and historical reference.

**Reminders**

- **Type:** Table
- **Function:** This tile serves as a detailed record of all reminders created by agents, which is useful for tracking, analysis, and historical reference.
