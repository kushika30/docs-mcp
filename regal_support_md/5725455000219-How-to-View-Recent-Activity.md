# How to View Recent Activity

### When you first start sending events to Regal, it's helpful to see that those events are arriving with the right information.

You can view the last 1,000 events you've sent in the last 7 days from the **Recent Activity** page. As your events arrive, you can click the **"Refresh Data"** button to see them in real-time.

![Recent Activity](https://files.readme.io/fae927c-recen.png "recen.png")

Once you start using Regal, you can also see all of the events Regal publishes from this page, including SMS and Task events.

Click into the Event Details to see the properties of each event.

![Event Details](https://files.readme.io/fa8068d-event.png "event.png")
