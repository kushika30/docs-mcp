# How to See Which Journeys a Campaign is Used In

### The same campaign can be triggered by multiple journeys.

When trouble shooting journeys or before editing a campaign, it's useful to first see which journeys that campaign is used in.

To see which journeys a campaign is used in:

1. From the Campaigns page, select a campaign.
2. Scroll to the bottom of the campaign detail page.
3. You'll see a component called **"Campaign Used In Follow Journeys"**
4. Click the arrow to expand to see the list of journeys. Journeys are ordered by created date. Older journeys are listed at the top. Newer journeys at the bottom.
5. Click on any of the journeys to view that journey in a new tab.

![Screen_Shot_2022-10-03_at_7.34.18_PM.png](https://support.regal.ai/hc/article_attachments/9328270825627/Screen_Shot_2022-10-03_at_7.34.18_PM.png)

> #### 🚧  Which journeys are listed?
>
> The list of journeys includes journeys in both Draft and Live states. Additionally, the journey will be listed even if the campaign is included in a node that has been dragged onto the journey canvas, but not hooked up to the journey flow.
