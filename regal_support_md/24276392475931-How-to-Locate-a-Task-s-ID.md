# How to Locate a Task's ID

A task's ID is a unique identifier for a task which represents a pending or completed interaction with a contact. The ID can be used for many things, such as locating information in reporting or when troubleshooting a problem with Regal's support team. Task IDs always start with the letters "WT" and agents can locate a task's ID in three places.

# Option One: Locating the ID of a Task You Are Currently Working On

From the agent desktop, click Campaign Info

![ScreenRecording2024-03-28at1.55.09PM-ezgif.com-video-to-gif-converter.gif](https://support.regal.ai/hc/article_attachments/24276392463387)

# Option Two: Locating the ID of an Upcoming or Past Task

Navigate to the tasks page, locate the task you're looking for, then scroll to the right.

![ezgif-7-6fa0853436.gif](https://support.regal.ai/hc/article_attachments/24276564388891)

# Option Three: Locating the ID of Past Task

1. Navigate to the audience page
2. Search for the contact the the task belongs to
3. Open the profile and switch the view to Event History![Screenshot 2024-03-28 at 1.57.48 PM.png](https://support.regal.ai/hc/article_attachments/24276647635739)
4. Locate an event that corresponds with the task you're looking for and copy the task ID from the task details![Screenshot 2024-03-28 at 1.58.32 PM.png](https://support.regal.ai/hc/article_attachments/24276376747419)
