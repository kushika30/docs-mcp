# How to Create Outbound Call Tasks

### The Create Call Task block creates an Outbound Call task in the queue for an agent to handle.

You must first [create the Call campaign](https://support.regal.ai/hc/en-us/articles/5721270757275) from the campaign builder. Then, once you've selected the Create Call Task block in the journey builder, you'll see a dropdown of call campaigns to choose from.

![Create Call Task](https://files.readme.io/c053343-call.png "call.png")

> ### 📘 Call Campaigns Must Be in Ready State to Send
>
> A Call campaign must be Saved as Ready in order to trigger it through a journey. If it is in Draft state, the call task will not be created and the next block in the journey will be processed.
